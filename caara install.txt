# goweslurmalangg cara clone agar sama semua

git clone https://github.com/sakhiasyam18/goweslurmalangg.git
cd goweslurmalangg
composer i

Nyalakan xampp
Buka phpMyAdmin
Klik tab “Databases”
Buat baru database kosongan:
iniajagoweslurmalangoktober

Lalu ubah .env:
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=iniajagoweslurmalangoktober
DB_USERNAME=root
DB_PASSWORD=

dan jalankan:
php artisan key:generate
php artisan migrate:fresh --seed
php artisan serve
http://localhost:8000/

Hasilnya: database otomatis terisi semuanya
project langsung sama persis dengan punyamu.
